# Arcade Floor — Plan Your Space

An interactive onboarding wizard that turns a customer's room into a usable game-floor
estimate (about 1 game per 70 sq ft). Start three ways — enter a **rectangle**, **draw** an
irregular shape, or **upload a plan/photo** and trace it — then highlight the usable area and
see how many games fit, with confidence grading and validation flags as you go.

Static, browser-only — open `index.html`. Uploaded images stay in your browser (never
uploaded anywhere). Proof of concept.
